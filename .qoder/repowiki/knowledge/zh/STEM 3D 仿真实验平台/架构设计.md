- **路由与页面层**：采用 Next.js App Router 架构，`src/app/experiments/[id]/page.tsx` 作为统一入口，通过动态导入 `src/experiments/*-page.tsx` 组件实现业务逻辑隔离。
- **仿真核心层**：遵循“Page-Scene”分离模式，`*-page.tsx` 负责 React 状态管理（参数控制、UI 交互），`*-scene.tsx` 封装 `@react-three/fiber` 画布逻辑，利用 `useFrame` 钩子执行高频物理迭代（如 RK4 积分）并驱动 3D 场景渲染。
- **通用 UI 组件库**：`src/components/experiment-ui` 提供标准化的 `ExperimentContainer`、`FloatingControlPanel` 及 `SimulationController`，确保所有实验具备一致的沉浸式全屏体验、悬浮控制面板及实时数据展示能力。
- **数据与工具层**：`src/data/experiments.ts` 集中管理实验元数据（分类、难度、描述）；`src/utils/physics.ts` 封装纯函数形式的物理定律计算（如单摆周期、气体状态方程），实现仿真逻辑与视图解耦。
