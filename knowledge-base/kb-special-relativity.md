# PhysLab 3D · 狭义相对论实验室说明

## 实验简介

狭义相对论实验室是 PhysLab 3D 平台中的相对论类交互实验，路由为 `special-relativity`。

以高速宇宙飞船为模型，演示爱因斯坦狭义相对论中的典型效应：长度收缩、时间膨胀与相对论质量（或能量）随速度升高而剧增。当速度趋近光速时，洛伦兹因子 γ 急剧增大——这也是有静质量的物体无法被加速到光速的原因之一。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 首页进入「狭义相对论实验室」（路由 `special-relativity`）。
2. 使用 **控制 / 数据 / 评论**；场景可拖拽旋转、滚轮缩放、右键平移。
3. 底部：播放/暂停、重置、速度。重置通常会把飞船速度拉回静止附近。
4. 「查看实验详情」进入原理页。

### 可调参数（控制面板 · 相对论参数）

| 控件名 | 作用 | 范围（平台） |
|--------|------|----------------|
| **飞船速度 v** | 相对光速的速度 | 约 0–99.5 % c（内部上限约 0.995c） |

本实验**主要就这一个连续参数**；其余靠预设按钮快速跳转。

### 快速预设按钮

- **静止**
- **0.5c**
- **0.9c**
- **0.95c**
- **0.99c**
- **0.995c**

建议教学顺序：静止 → 0.5c（几乎无感）→ 0.9c → 0.99c → 0.995c（效应陡增）。

### 数据面板会显示什么

- velocity（% c）、gamma（γ）、length（长度百分比）、mass（质量倍数）
- 卡片：**时间膨胀**、**长度收缩**、**相对论质量**（含公式与说明）
- 场景中常有 γ–v/c 曲线辅助理解

### 推荐操作流程

1. 点「静止」，确认飞船长度与时钟为基准状态。
2. 点「0.5c」，说明低速效应很小。
3. 点「0.9c」，观察长度缩短与 γ 明显增大。
4. 再点「0.99c」「0.995c」，强调接近光速时 γ 急剧上升、长度趋于很小、质量/能量发散趋势。
5. 也可慢慢拖动「飞船速度 v」滑块，盯着数据面板里的 γ 与 length。
6. 提醒学生：长度收缩只沿运动方向；垂直方向不变。

### 助手回答用户时应强调

- 能调：飞船速度 v（滑块）+ 上述快速预设 + 播放速度。
- 没有单独的“质量滑块/长度滑块”——它们是由 v 算出来的观测量。
- 想看“接近光速有多夸张”：直接让用户点 **0.99c / 0.995c**。

## 核心公式

### 洛伦兹因子

γ = 1 / √(1 − v²/c²)

- v：相对速度
- c：真空光速
- 当 v → c 时，γ → ∞

### 时间膨胀

Δt' = γ · Δt₀

运动时钟相对地面观测变慢。直观说法：飞船上流逝 1 秒，对应地面约 γ 秒。

### 长度收缩

L' = L₀ / γ

仅沿相对运动方向收缩。UI 中长度百分比约为 100/γ。

### 相对论质量（教学表述）

m = γ · m₀

或等价地强调相对论能量 E = γ m₀ c²。v → c 时继续加速所需能量趋于无穷。

## 参数调节建议（给助手与学生）

- 低速几乎无感：先对比「静止」与 0.5c。
- 想看显著效应：依次使用 0.9c → 0.99c → 0.995c，观察 γ 与长度/质量剧变。
- 一次只改速度；结合 γ–v/c 曲线理解“接近光速时效应陡增”。
- 强调：长度收缩只发生在运动方向，垂直方向不变。

## 核心概念

- **光速不变原理**：真空中光速对所有惯性系相同。
- **时间膨胀**：运动时钟变慢（相对某一惯性系的观测）。
- **长度收缩**：运动物体沿运动方向测得更短。
- **洛伦兹因子 γ**：统一刻画上述效应强弱的关键因子。
- **无法达到光速**：有静质量物体加速到 c 需要无穷能量。

## 简要历史

- 1905：爱因斯坦提出狭义相对论，以光速不变与相对性原理为基础。
- 此后：时间膨胀、质能关系等在粒子物理与精密实验中得到广泛验证。

## 实际应用关联

- GPS 卫星需考虑相对论时间修正（狭义与广义效应均有贡献）。
- 高能加速器中粒子寿命变长（时间膨胀）是日常运行事实。
- 核反应与粒子物理广泛使用 E = mc² 及相关能量关系。

## 常见问题（FAQ）

**问：为什么接近光速时飞船看起来特别短？**  
答：沿运动方向 L' = L₀/γ，v 越接近 c，γ 越大，观测长度越小。

**问：「γ 倍变慢」是什么意思？**  
答：相对地面惯性系，飞船上时钟按因子 γ 变慢，即 Δt' = γ Δt₀。

**问：为什么有质量的物体到不了光速？**  
答：v → c 时 γ → ∞，继续提高速度所需能量趋于无穷大。

**问：本实验适合回答哪些问题？**  
答：γ 的计算与趋势；长度收缩与时间膨胀公式；预设速度下的数量级对比；无法超光速的教学解释。
