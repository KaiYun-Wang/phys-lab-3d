# PhysLab 3D · 广义相对论 · 史瓦西黑洞说明

## 实验简介

广义相对论 · 史瓦西黑洞是 PhysLab 3D 平台中的引力/相对论交互实验，路由为 `general-relativity`。

在广义相对论中，引力体现为时空弯曲，自由物体沿测地线运动。本实验模拟不旋转的史瓦西黑洞：可发射测试粒子观察轨道、发射光子观察引力透镜；蓝色网格（Flamm 嵌入示意）把时空弯曲画成漏斗。与牛顿「引力轨道」不同：近视界会出现进动、更强的光偏折与引力红移等效应。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 首页进入「广义相对论 · 史瓦西黑洞」（路由 `general-relativity`）。
2. 使用 **控制 / 数据 / 评论**；**拖拽旋转、滚轮缩放、右键平移** 非常重要（场景复杂）。
3. 底部：播放/暂停、重置、速度（本实验默认速度往往偏快，可自行调低）。
4. 可能弹出引导「你在看什么？」——介绍网格/视界/吸积盘/粒子/光子；可关闭后继续。
5. 「查看原理说明」进入详情页。

### 推荐操作流程

1. 打开「时空弯曲网格」，旋转视角看漏斗状弯曲。
2. 只调 **质量 M**，看视界大小变化。
3. 点预设「坠入」再「发射粒子」，观察落入视界的轨迹。
4. 点预设「逃逸」再发射，对比能否飞离。
5. 手动把 r 调到小于约 3rs（ISCO）附近，体会稳定圆轨道难维持。
6. 调小 **碰撞参数 b**，「发射光子」，看偏折变大；过大偏折可能被俘获。
7. 到「数据」看红移 z、偏折角等；关掉部分图层减轻干扰。

### 助手回答用户时应强调

- 能调：质量 M；粒子的 r / 切向速度 / 径向速度 + 发射；光子 b + 发射；各显示图层；播放速度。
- 不是“自动一直掉粒子”——通常要点 **发射粒子 / 发射光子**。
- 想看透镜：主推调 **b**；想看轨道：主推预设坠入/逃逸或改切向速度。
- 与牛顿轨道实验不同：本实验是广义相对论测地线。

## 核心公式

### 史瓦西半径

rs = 2GM / c²

在几何单位（G = c = 1）下常写为 rs = 2M。质量越大，视界越大。

### 史瓦西度规（赤道面、示意）

ds² = −(1 − rs/r) c² dt² + (1 − rs/r)⁻¹ dr² + r² dφ²

r = rs 为事件视界（坐标奇点处的视界面）。

### 引力红移

z = 1 / √(1 − rs/r) − 1

越靠近视界，从该处发出的光到达远处观测者时红移越大。

### 重要特征半径（教学常用）

- 光子球：约 1.5 rs（不稳定的光子圆轨道）
- 最内稳定圆轨道 ISCO：约 3 rs（小于此半径的圆轨道在真实 GR 中不稳定）

弱场光偏折量级可示意写作与 rs/b 相关（b 为碰撞参数）；近视界需用完整测地线数值结果。

## 参数调节建议（给助手与学生）

- 先打开时空弯曲网格，建立“漏斗”直觉。
- 粒子：用「坠入」「逃逸」预设入手；r 小于约 3 rs（ISCO）时更难维持稳定圆轨道。
- 光子：减小碰撞参数 b，偏折更强，甚至可能被俘获（轨迹表现会明显变化）。
- 改质量 M 后 rs 与相关尺度一起缩放；画面为教学加速，可用仿真速度再加快观察。

## 核心概念

- **时空弯曲与测地线**：自由运动遵循弯曲时空中的“直线”。
- **事件视界**：光也无法从内部逃逸的临界面（r = rs）。
- **光子球 / ISCO**：光子与物质轨道的特征半径。
- **引力透镜**：光线被时空弯曲偏折，形成弧线路径。
- **引力红移**：深引力势中发出的光频率降低。
- **与牛顿差异**：远视界近似牛顿，近视界出现进动、更强偏折等。

## 简要历史

- 1915：爱因斯坦提出广义相对论。
- 1916：史瓦西给出球对称真空解，成为最简单（不旋转）黑洞模型。
- 现今天文观测已支持包括黑洞阴影、引力波等与强引力相关的现象。

## 实际应用关联

- 黑洞吸积盘与喷流的理论研究与观测解释。
- 引力透镜用于天文学质量测量与宇宙学。
- GPS 等精密授时需计入地球引力导致的广义相对论时间修正。

## 常见问题（FAQ）

**问：粒子轨道和光子路径有何不同？**  
答：有质量粒子可形成束缚、坠入等测地线轨道；光子无静质量，路径由零测地线描述，表现为引力偏折（透镜）。

**问：为什么离视界还不远就容易“掉进去”？**  
答：在 ISCO（约 3 rs）以内，稳定圆轨道难以维持，往往螺旋落入；远处则取决于切向速度是否足够、是否有向内径向速度。

**问：和站内牛顿风格的引力轨道实验差在哪？**  
答：牛顿用平方反比力；本实验用爱因斯坦测地线。远视界两者接近，近视界差异巨大（进动、光偏折、红移等）。

**问：本实验适合回答哪些问题？**  
答：rs 与质量关系；视界/光子球/ISCO；引力红移公式；透镜偏折随 b 的变化；GR 与牛顿的定性差异。
