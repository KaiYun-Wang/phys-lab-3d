# PhysLab 3D · 伯努利原理（文丘里管）说明

## 实验简介

伯努利原理（文丘里管）是 PhysLab 3D 平台中的流体力学交互实验，路由为 `bernoulli-venturi`。

文丘里管是测量流速/流量的经典装置，也是演示伯努利原理的理想模型：管道截面变小时，流速增大、静压降低；截面扩大时则相反。本实验用 3D 管段与测压管液面高低，直观表现压强差 ΔP。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 首页进入「伯努利原理（文丘里管）」（路由 `bernoulli-venturi`）。
2. 使用 **控制 / 数据 / 评论**；场景可拖拽旋转、滚轮缩放、右键平移。
3. 底部：播放/暂停、重置、速度。重置常会恢复默认：v₁≈2 m/s、A₂/A₁≈0.5、介质=水。
4. 「查看实验详情」看原理说明。

### 可调参数（控制面板 · 流体参数）

| 控件名 | 作用 | 范围（平台） |
|--------|------|----------------|
| **入口流速 v₁** | 管道入口速度 | 约 0–5 m/s |
| **截面积比 A₂/A₁** | 喉部（或第二段）相对入口截面积 | 约 0.2–2.0 |

判读：
- A₂/A₁ **< 1**：收缩管 → 通常 v₂↑、P₂↓，对应测压管液面更低
- A₂/A₁ **> 1**：扩张管 → 相反趋势
- A₂/A₁ **= 1**：等径，两侧液面接近齐平

### 流体介质

| 选项 | 含义 |
|------|------|
| **水** | 密度约 1000 kg/m³ |
| **甘油** | 密度约 1260 kg/m³ |

面板会显示当前密度 ρ。同速度下密度越大，|ΔP| 往往越大。

### 数据面板会显示什么

- v1、v2、rho、deltaP（单位分别为 m/s、m/s、kg/m³、Pa）
- 连续性方程 / 伯努利方程 / 压强差 / 测压说明文字
- 状态提示：收缩管 / 扩张管 / 等径管及液面高低解读

### 推荐操作流程

1. 默认收缩管（A₂/A₁≈0.5）播放，看喉部流速与右侧测压管液面。
2. **只增大 v₁**，观察 ΔP 与液面差变大。
3. 把 A₂/A₁ 调到 **> 1**，对比扩张管液面变化。
4. 把 A₂/A₁ 调到 **1**，确认等径时两侧接近齐平。
5. 介质在水 ↔ 甘油间切换，体会密度对 ΔP 的影响。
6. 到「数据」面板对照 v₂ = v₁/(A₂/A₁) 与 ΔP = ½ρ(v₂²−v₁²)。

### 助手回答用户时应强调

- 能调：入口流速 v₁、截面积比 A₂/A₁、介质（水/甘油）、播放速度。
- 看压强：主要看测压管液面高低 + 数据里的 deltaP。
- 想证明“流速大静压小”：保持收缩比 < 1，增大 v₁ 最直观。

## 核心公式

### 连续性方程（不可压缩、稳定流）

A₁v₁ = A₂v₂

因此：v₂ = v₁ / (A₂/A₁)

截面越小，流速越大。

### 伯努利方程（水平、理想流体近似）

P₁ + ½ρv₁² = P₂ + ½ρv₂²

### 压强差

ΔP = P₁ − P₂ = ½ρ(v₂² − v₁²)

- ρ：流体密度
- 收缩管（v₂ > v₁）：ΔP > 0，即 P₂ < P₁，喉部静压更低
- 扩张管（v₂ < v₁）：ΔP < 0，下游静压相对更高

测压管液面高度差与压强差成正比，可用于定性读取 ΔP。

## 参数调节建议（给助手与学生）

- 想看收缩效应：令 A₂/A₁ < 1，观察 v₂ 升高、P₂ 降低、对应测压管液面更低。
- 想看扩张效应：令 A₂/A₁ > 1，观察右管液面相对升高。
- 想放大压差：增大 v₁，或换用密度更大的甘油。
- 等径管（A₂/A₁ = 1）时两侧液面应接近齐平，便于校准直觉。

## 核心概念

- **伯努利原理**：沿流线，流速与静压在一定条件下呈反比关系（水平稳定流的常用课堂表述）。
- **连续性方程**：体积流量守恒导致截面与流速反比。
- **静压 / 动压**：静压由测压体现；动压与 ½ρv² 相关。
- **文丘里效应**：收缩段加速、降压的工程现象。

## 实际应用关联

- 文丘里流量计测量管道流量。
- 喷雾器、化油器等利用低压吸入液体或气体。
- 机翼升力的课堂解释常与伯努利效应联系（完整分析还需环量与压强分布）。

## 常见问题（FAQ）

**问：为什么喉部流速更大，压强反而更低？**  
答：截面变小 → 由连续性知 v₂ 更大；伯努利要求动压升高时静压降低，以保持能量关系。

**问：右管液面比左管高或低分别说明什么？**  
答：更低通常对应收缩管（P₂ < P₁）；更高对应扩张管；齐平接近等径情况。

**问：换成甘油会有什么变化？**  
答：密度 ρ 更大，在相近速度下 |ΔP| 更大，液面差往往更明显。

**问：本实验适合回答哪些问题？**  
答：A–v 关系；ΔP 公式；收缩/扩张判读；密度对压差的影响；文丘里装置用途。
