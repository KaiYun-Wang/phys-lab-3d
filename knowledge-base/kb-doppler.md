# PhysLab 3D · 多普勒效应说明

## 实验简介

多普勒效应是 PhysLab 3D 平台中的声学/波动类交互实验，路由为 `doppler`。

当波源与观察者发生相对运动时，观察者接收到的频率会改变：波源靠近时波前被压缩，频率升高（常称蓝移）；远离时波前被拉伸，频率降低（常称红移）。本实验用可运动的声源、波前与实时数据面板，直观演示这一现象。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 首页进入「多普勒效应」（路由 `doppler`）。
2. 使用 **控制 / 数据 / 评论** 面板；场景可拖拽旋转、滚轮缩放、右键平移。
3. 底部有播放/暂停、重置、速度条（界面统一提供；本实验主体参数以控制面板为准）。
4. 「查看实验详情」进入原理页。

### 可调参数（控制面板 · 波参数）

| 控件名 | 作用 | 范围（平台） |
|--------|------|----------------|
| **源频率 (f₀)** | 声源固有频率 | 约 0.5–5 Hz |
| **源速度 (vₛ)** | 声源运动快慢 | 约 0–15 m/s |
| **波速 (v)** | 介质中波速 | 约 5–20 m/s |

### 声源运动

| 控件 | 作用 |
|------|------|
| **模式：自动 / 手动** | 自动=声源往返振荡；手动=自己定方向 |
| **方向**（仅手动） | 滑块约 −1…1，对应向左 / 静止 / 向右 |
| **观察者位置** | 观察者在场景中的 x 位置，约 −20…20 m |

### 显示选项

- **显示波前**：开关波前圆环/波面可视化，便于看压缩与拉伸。

### 数据面板会显示什么

- 源频率、观测频率、多普勒比、马赫数、波速
- **频移类型**：蓝移（靠近）/ 红移（远离）/ 无频移（静止）
- 可能出现超音速警告（马赫数 ≥ 1）以及频率升高/降低百分比

### 推荐操作流程

1. 先用 **自动** 模式、打开「显示波前」，看声源来回时波前疏密变化。
2. 打开「数据」，对照观测频率与蓝移/红移标签。
3. 切到 **手动**，固定「向右」或「向左」，对比靠近与远离时 f' 高低。
4. **只增大源速度 vₛ**，看多普勒比与频移幅度变大。
5. 移动「观察者位置」，体会相对几何关系对频移的影响。
6. 将 vₛ 调接近波速 v，观察马赫数接近 1 时的提示（注意公式在接近波速时会极端）。

### 助手回答用户时应强调

- 能调：f₀、vₛ、v、自动/手动、方向、观察者位置、显示波前。
- 想演示蓝移：让声源朝观察者方向运动（手动锁定方向最清楚）。
- 想演示红移：让声源远离观察者。
- 波前开关不影响公式，只影响可视化。

## 核心公式

### 运动声源、静止观察者（经典形式）

f' = f₀ · v / (v ± vₛ)

- f₀：源频率
- f'：观察者接收频率
- v：波在介质中的传播速度
- vₛ：声源速度
- 靠近观察者时分母取 (v − vₛ)；远离时取 (v + vₛ)

含义：源朝观察者靠近 → f' > f₀；远离 → f' < f₀。

### 马赫数

Ma = |vₛ| / v

当 Ma 接近或超过 1 时，源速达到/超过波速，经典多普勒公式会极端放大，平台会给出超音速相关提示。

## 参数调节建议（给助手与学生）

- 想让频移更明显：增大源速度 vₛ，或把观察者放在声源运动路径的一侧。
- 想固定对比蓝移/红移：用手动模式锁定方向。
- 想看波前压缩：打开「显示波前」，并适当提高 vₛ。
- 接近超音速时注意 vₛ 不要长期等于或大于 v，以免数值失真；可先看数据面板的多普勒比。

## 核心概念

- **多普勒效应**：相对运动导致观测频率改变。
- **蓝移 / 红移**：频率升高 / 降低的形象说法（光学中对应波长变短/变长）。
- **波前**：波上相位相同点的连线；运动会使波前疏密不均。
- **马赫数**：源速与波速之比，刻画是否接近超音速。

## 实际应用关联

- 交警测速雷达利用电磁波多普勒测速。
- 医学超声利用血流多普勒评估血管与心脏功能。
- 天文上星系光谱红移与宇宙膨胀相关。
- 气象雷达可监测降水与气流运动。

## 常见问题（FAQ）

**问：蓝移和红移怎么区分？**  
答：源朝观察者靠近 → 频率升高（蓝移）；远离 → 频率降低（红移）；相对静止 → 无频移。

**问：马赫数 ≥ 1 意味着什么？**  
答：源速达到或超过波速。此时波前严重堆积，经典多普勒公式不再适用日常低频移直觉。

**问：自动和手动模式有何区别？**  
答：自动模式下声源来回运动，频移随时间周期变化；手动模式可固定方向，便于对照公式。

**问：本实验适合回答哪些问题？**  
答：f' 与 f₀、v、vₛ 的关系；蓝移/红移判据；波前形态与相对运动；马赫数的含义。
