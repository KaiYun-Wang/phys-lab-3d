# PhysLab 3D · 横波与纵波说明

## 实验简介

横波与纵波是 PhysLab 3D 平台中的波动类交互实验，路由为 `wave-mechanics`。

波在介质中传播能量，质点本身并不随波整体迁移。按振动方向与传播方向的关系，可分为：横波（振动方向垂直于传播方向，出现波峰/波谷）与纵波（振动方向平行于传播方向，出现密区/疏区）。本实验默认左右分屏对比，参数联动，便于直观区分两类波动。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 首页进入「横波与纵波」（路由 `wave-mechanics`）。
2. 切换 **控制 / 数据 / 评论** 面板；三维场景可拖拽旋转、滚轮缩放、右键平移。
3. 底部工具栏：播放/暂停、重置、**速度**（约 0.1×–3×）。
4. 首次进入可有教学引导（「这是横波 / 这是纵波」等），可「跳过」或「下一步 / 开始探索」。
5. 点「进入实验原理」看文字说明；「重新观看教学引导」可再看引导。

### 可调参数（控制面板 · 波动参数）

| 控件名 | 作用 | 范围（平台） |
|--------|------|----------------|
| **频率 f** | 振动快慢 | 约 0.5–4 Hz |
| **振幅 A** | 振动幅度 | 约 0.2–2 |
| **波长 λ** | 相邻同相点距离 | 约 2–8 m |
| **波速 v** | 传播速度；与 f、λ 联动（v = fλ） | 约 1–32 m/s |

注意：改 **波速 v** 时，平台常会按 v = fλ **反推波长 λ**，不要以为三个量完全独立。

### 视图模式

桌面：**对比**（左右分屏） / **横波** / **纵波** / **叠加**。  
手机顶部标签：**横波 | 纵波**。

### 数据面板会显示什么

- 时间、频率、波长、波速、波数 k、角频率 ω
- 横波 ymax / ymin；纵波 ρmax / ρmin
- 公式：横波 y = A sin(kx − ωt)；纵波 Δx = A sin(kx − ωt)
- 若点击了某个质点：显示该点位移 y(t) 或压缩度 ρ(t) 及曲线

### 推荐操作流程

1. 用默认「对比」模式播放，看左侧横波峰谷与右侧纵波疏密。
2. **只增大振幅 A**，看振动更明显。
3. **只增大频率 f**（注意波速可能联动），看振荡加快。
4. 调节波长 λ 或波速 v，理解 v = fλ。
5. 暂停，观察两侧相位线是否同步。
6. 点击画面中的质点，到「数据」面板看该点时间序列。
7. 试「叠加」模式，对比振动方向差异。

### 助手回答用户时应强调

- 能调：f、A、λ、v，以及视图模式、播放速度。
- 联动：改 v 往往会改 λ；讲解时提醒一次只改一个量。
- 可点击质点做局部观测；可左右分屏对比。

## 核心公式

### 横波位移

y = A · sin(kx − ωt)

质点沿垂直于传播方向振动，形成波峰与波谷。

### 纵波位移

Δx = A · sin(kx − ωt)

质点沿传播方向振动，形成疏密相间的区域。

### 波速关系

v = fλ = ω / k

- f：频率
- λ：波长
- ω：角频率
- k：波数

### 压缩度（纵波）

ρ = d₀ / dᵢ

- ρ > 1：密区
- ρ < 1：疏区

## 参数调节建议（给助手与学生）

- 想让峰谷/疏密更明显：适当增大振幅 A。
- 想在视野里看到更多周期：减小波长 λ，或增大频率 f。
- 想对比相位：暂停仿真，使用「对比」模式观察两侧相位线。
- 一次只改一个参数；改波速 v 时注意波长 λ 可能随之变化。

## 核心概念

- **横波**：振动方向垂直于传播方向（如绳波、电磁波）。
- **纵波**：振动方向平行于传播方向（如声波、弹簧疏密波）。
- **波峰 / 波谷**：横波上位移最大与最小的位置。
- **密区 / 疏区**：纵波中介质局部压缩与稀疏的区域。
- **相位**：描述振动步调；同相位线表示两侧对应质点同步振动。

## 实际应用关联

- 声波在空气中的传播属于纵波。
- 弦乐器弦上的振动可近似为横波传播。
- 地震波同时包含横波（S 波）与纵波（P 波）成分。

## 常见问题（FAQ）

**问：为什么改频率后波形好像“跑得更快”？**  
答：若波长大致不变，则 v = fλ 增大，传播更快。也可直接调波速滑块，观察与波长的联动。

**问：纵波颜色疏密代表什么？**  
答：对应局部压缩度 ρ。密区 ρ > 1，疏区 ρ < 1。

**问：叠加模式在看什么？**  
答：同一套参数下把横波与纵波叠在一起，便于对比振动方向差异。

**问：本实验适合回答哪些问题？**  
答：横波与纵波的区别；v、f、λ 关系；疏密与峰谷的含义；相位同步观察方法。
