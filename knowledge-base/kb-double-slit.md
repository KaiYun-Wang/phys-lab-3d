# PhysLab 3D · 双缝实验说明

## 实验简介

双缝实验（杨氏双缝干涉实验）是 PhysLab 3D 平台中的光学/量子类交互实验，路由为 `double-slit`。

它展示了波粒二象性：当光（或微观粒子）通过两条间距很近的狭缝时，会在远处屏幕上形成明暗相间的干涉条纹。即使粒子一个一个通过，长时间累积后仍会出现干涉图样。

该实验最初由托马斯·杨于 1801 年完成，帮助确立了光的波动理论；在现代量子力学中，它也常用来说明叠加原理与测量问题。

## 在本平台怎么操作

1. 从首页进入「双缝实验」。
2. 在左侧控制面板调节波长、缝距等参数。
3. 观察屏幕（或接收屏）上干涉条纹疏密如何变化。
4. 需要原理说明时可打开实验详情页，或向 AI 助手提问。

## 核心公式

### 条纹间距

Δy = λL / d

- λ：波长
- L：双缝到屏幕的距离
- d：两缝间距
- Δy：相邻亮条纹（或相邻暗条纹）之间的距离

含义：波长越大，或缝距越小，或屏距越远，条纹越疏（间距越大）。

### 光程差

Δr = d · sin(θ)

两缝到观察点的路径长度之差。小角度近似下可用 θ ≈ y/L 简化。

### 干涉条件

- 相长干涉（亮纹）：Δr = mλ（m = 0, 1, 2, …）
- 相消干涉（暗纹）：Δr = (m + 1/2)λ

## 参数调节建议（给助手与学生）

- 想让条纹更密：增大缝距 d，或减小波长 λ，或减小屏距 L。
- 想让条纹更疏：减小缝距 d，或增大波长 λ，或增大屏距 L。
- 一次只改一个参数，更容易看清因果关系。
- 若画面上看不清条纹，先检查缝距是否过大、波长是否过短导致条纹挤在一起。

## 核心概念

- **波粒二象性**：光和物质既可表现波动性，也可表现粒子性，与观测方式有关。
- **干涉**：两列相干波叠加时，相位相同处加强（亮），相反处减弱（暗）。
- **叠加态**：量子系统在测量前可处于多种可能状态的叠加。
- **衍射**：波通过与波长可比拟的开口时会扩散，双缝实验中常与干涉同时出现。

## 简要历史

- 1801：托马斯·杨完成经典双缝实验，支持光的波动说。
- 1909：G.I. 泰勒用极弱光（接近单光子量级）仍观察到干涉。
- 1961：克劳斯·荣松完成电子双缝实验。
- 此后：中子、原子乃至大分子也陆续演示出干涉现象。

## 实际应用关联

- 全息术：利用干涉记录与再现三维信息。
- 光谱分析：干涉与衍射是光谱仪器的重要原理基础。
- 光学镀膜：增透膜等利用干涉调控反射与透射。
- 相控阵：用可控相位差形成定向波束，思想与干涉相关。

## 常见问题（FAQ）

**问：为什么条纹间距突然变大了？**  
答：多半是波长变大、缝距变小，或屏距变大。对照公式 Δy = λL/d 逐项检查。

**问：关闭一条缝后条纹会怎样？**  
答：不再是双缝干涉图样，通常变为单缝衍射（或更接近单缝光照）分布，中央亮区与旁瓣形态会改变。

**问：这和「观察会破坏干涉」有什么关系？**  
答：若通过测量获知粒子究竟走了哪条缝，路径信息出现后，干涉条纹往往会消失；这是量子测量与互补性的典型课堂讨论点。平台演示侧重波动干涉与参数关系，深入诠释可结合教材。

**问：本实验适合回答哪些问题？**  
答：条纹疏密与 λ、d、L 的关系；亮暗纹条件；杨氏实验历史意义；与波粒二象性相关的入门概念。
