# PhysLab 3D · 双缝实验说明

## 实验简介

双缝实验（杨氏双缝干涉实验）是 PhysLab 3D 平台中的光学/量子类交互实验，路由为 `double-slit`。

它展示了波粒二象性：当光（或微观粒子）通过两条间距很近的狭缝时，会在远处屏幕上形成明暗相间的干涉条纹。即使粒子一个一个通过，长时间累积后仍会出现干涉图样。

该实验最初由托马斯·杨于 1801 年完成，帮助确立了光的波动理论；在现代量子力学中，它也常用来说明叠加原理与测量问题。

## 在本平台怎么操作（给 AI / 学生的完整教程）

### 进入方式

1. 打开 PhysLab 3D 首页，点击「双缝实验」（路由 `double-slit`）。
2. 页面顶部有「返回」；右侧/底部可切换 **控制**、**数据**、**评论** 三个面板。
3. 三维场景通用操作：**拖拽旋转 · 滚轮缩放 · 右键平移**。
4. 底部工具栏：▶/⏸ 播放暂停、↺ 重置、**速度**（约 0.1×–3×）。

### 可调参数（控制面板 · 量子参数）

| 控件名 | 作用 | 范围（平台） |
|--------|------|----------------|
| **缝间距 (d)** | 两缝中心距；影响条纹疏密 | 约 0.5–5 mm |
| **缝宽 (a)** | 单缝宽度；影响衍射包络 | 约 0.1–1 mm |
| **发射速率** | 单位时间发射粒子数 | 约 1–10 个/秒 |

说明：本实验界面**没有波长滑块**，波长固定约 **500 nm**。向学生讲解条纹公式时可用理论波长，但平台上手动调节的是 d、a 与发射速率。

### 显示选项

| 控件 | 作用 |
|------|------|
| **观测开启 / 波动模式** | 切换粒子行为（观测/坍缩示意）与波干涉（叠加态）显示 |
| **显示粒子** | 是否显示飞行粒子 |

点击「查看实验详情」可进入原理说明页。

### 数据面板会显示什么

- 波长（nm）、条纹间距（mm）、缝间距（mm）、粒子数
- 波粒二象性提示文案（观测开/关时不同）
- 强度公式示意：`I(y) = cos²(…) · sinc²(…)`

### 推荐操作流程

1. 先保持默认参数，播放，看屏幕上条纹如何随粒子累积出现。
2. **只增大缝间距 d**，观察条纹是否变密；再减小 d，看是否变疏。
3. 调节缝宽 a，对比衍射包络宽窄变化。
4. 提高发射速率，加快累积；需要仔细看时降低仿真速度。
5. 切换「观测开启 / 波动模式」，对比粒子轨迹与干涉图样的不同展示侧重点。
6. 若条纹过密看不清：适当减小 d，或放慢速度、放大视角。

### 助手回答用户时应强调

- 能调：缝间距 d、缝宽 a、发射速率、观测/波动模式、显示粒子、播放速度。
- 不能调：波长（界面固定约 500 nm）。
- 想改条纹疏密：优先让用户动 **缝间距 d**。

## 核心公式

### 条纹间距

Δy = λL / d

- λ：波长
- L：双缝到屏幕的距离
- d：两缝间距
- Δy：相邻亮条纹（或相邻暗条纹）之间的距离

含义：波长越大，或缝距越小，或屏距越远，条纹越疏（间距越大）。

### 光程差

Δr = d · sin(θ)

两缝到观察点的路径长度之差。小角度近似下可用 θ ≈ y/L 简化。

### 干涉条件

- 相长干涉（亮纹）：Δr = mλ（m = 0, 1, 2, …）
- 相消干涉（暗纹）：Δr = (m + 1/2)λ

## 参数调节建议（给助手与学生）

- 想让条纹更密：增大缝间距 d（本平台主要手段）。
- 想让条纹更疏：减小缝间距 d。
- 理论公式中 λ、L 也会影响 Δy，但本实验 UI 不提供波长/屏距滑块。
- 一次只改一个参数，更容易看清因果关系。
- 条纹挤在一起：减小 d，或降低仿真速度、放大画面。

## 核心概念

- **波粒二象性**：光和物质既可表现波动性，也可表现粒子性，与观测方式有关。
- **干涉**：两列相干波叠加时，相位相同处加强（亮），相反处减弱（暗）。
- **叠加态**：量子系统在测量前可处于多种可能状态的叠加。
- **衍射**：波通过与波长可比拟的开口时会扩散，双缝实验中常与干涉同时出现。

## 简要历史

- 1801：托马斯·杨完成经典双缝实验，支持光的波动说。
- 1909：G.I. 泰勒用极弱光（接近单光子量级）仍观察到干涉。
- 1961：克劳斯·荣松完成电子双缝实验。
- 此后：中子、原子乃至大分子也陆续演示出干涉现象。

## 实际应用关联

- 全息术：利用干涉记录与再现三维信息。
- 光谱分析：干涉与衍射是光谱仪器的重要原理基础。
- 光学镀膜：增透膜等利用干涉调控反射与透射。
- 相控阵：用可控相位差形成定向波束，思想与干涉相关。

## 常见问题（FAQ）

**问：为什么条纹间距突然变大了？**  
答：在本平台上优先检查是否减小了缝间距 d。理论公式 Δy = λL/d 中 λ、L 也会影响，但界面不提供波长/屏距滑块（波长固定约 500 nm）。

**问：我能调节哪些参数？**  
答：缝间距 d、缝宽 a、发射速率；以及观测/波动模式、显示粒子、播放速度。不能调波长。

**问：关闭一条缝后条纹会怎样？**  
答：不再是双缝干涉图样，通常变为单缝衍射（或更接近单缝光照）分布，中央亮区与旁瓣形态会改变。

**问：这和「观察会破坏干涉」有什么关系？**  
答：若通过测量获知粒子究竟走了哪条缝，路径信息出现后，干涉条纹往往会消失；这是量子测量与互补性的典型课堂讨论点。平台演示侧重波动干涉与参数关系，深入诠释可结合教材。

**问：本实验适合回答哪些问题？**  
答：条纹疏密与 λ、d、L 的关系；亮暗纹条件；杨氏实验历史意义；与波粒二象性相关的入门概念。
